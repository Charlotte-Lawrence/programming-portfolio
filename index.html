Solar System

This project is an animated model of the solar system...

In this project, I created a 2D animated model of the solar system using Java.

I heavily used object-oriented principles for extensible and efficient code with each item inheriting from a custom SolarObject type, such as Sun and Planet and then Moon also inherits from Planet creating a hierarchy / inheritance tree.

The planets orbit the sun and there are moons which orbit specific planets. Although this project was a piece of cousework for my Advanced Programming module, I really enjoyed creating it and I was able to have enough time to add some extra features of my own.

Some features I added was being able to alter the speed of the model, or pausing it altogether, using keyboard keybinds. I also added an asteroid belt which consisted of randomly generated and sized asteroids, and also some stars for the background which was also randomly generated upon each load of the program.

This was created in Java but at some point I hope to extend the project further and potentially use Unity to remake it in 3D with correct planet textures.

Prison Escape Game

This project was a prison escape game for my A Level Computer Science project...

This project was for my Computer Science A Level Project.

It was written in C# and I created all of the graphics myself using photoshop. We were able pick our own idea completely so I picked a prison escape game inspired by The Escapists.

Some key features of the game include:

  • Animated player movements
  • AI for the guards - using Dijkstra's Shortest Path algorithm which uses a computer generated adjacency matrix (converting the map bounding boxes into nodes and edges, so when a new map is added the guards will automatically work for each map), and they also use a finite state machine to easily swap the guard's state (attack, roam, chase)
  • Camera viewport - follows the player and keeps them in the center of the frame until they go near the edges of the map
  • Usable items - money and keys
  • Strength and health systems
  • Attacking or bribing the guards
  • Escaping or dying

I really enjoyed creating this project and working with my group. Because of all of our hard work and dedication, our project was graded an A, and I achieved a first-class grade in that module.

Graphics Editor

This project was created for my university Computer Science Group Project module and is our take on a graphics editor such as Microsoft Paint...

This project was for my Computer Science Group Project module and was coded in Java and we used GitLab to streamline the code collaboration and deployment.

It is our take on a pixel-based graphics editor similar to Microsoft Paint. Some of the key features that I specifically worked on include:

  • Fill tool - using a flood fill algorithm with breadth first search
  • Layering functionality - similar to one in Adobe Photoshop and , other layer functions such as add, delete and renaming.
  • Select tool functionality - to move pixels dynamically
  • Export - the user can have the final image in a PNG file
  • File save and load - with our own program file extension so the user can continue editing their image.

I really enjoyed creating this project and working with my group and because of all of our hard work and dedication, our project was graded an A and I achieved a first class in that module.

Networking Applications

This project was a coursework element for my Computer Networks module and is written in Python...

This project was a coursework element for my Computer Networks module and is written in Python.

The goal of this project is to create or replicate certain commonly-used networking tools using pythons Socket library.

I was able to implement a ping function to calculate the delay when pinging a certain host. I also created traceroute and paris traceroute functionality to try to accurately get the path that a packet will go through across networks. I implemented paris traceroute specifically for accuracy when dealing with networks that have load balancers and it should remove most of the 'false links' that can sometimes be created with traditional traceroute.

In this project I also implemented a web server and proxy functions. The proxy function specifically handles HTTP get requests and also has caching so that when the client requests the same page it is obtained from the cache just like a typical web proxy does.

Scheduler

This project is a scheduling application for music festivals...

This project is a scheduling application for music festivals.

In this project, I was tasked to create a scheduler for a music festival.

The scheduler allows for the user to input an act name with their priority, and duration on stage. The user can then set a target time for the highest priority act to be at and they can also set a gap between acts so that they can move equipment etc.

The program also allows for a file upload for festivals with a large amount of acts and also a save to file option for ease of use. The text generated section also allows copying so the user can do that instead if they wish.
